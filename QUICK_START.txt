ASW Target Distribution Simulator - Cross-Platform Quick Start

This package contains standalone executables for Windows, macOS, and Linux.
No Python installation required on any platform!

PLATFORM-SPECIFIC EXECUTABLES:
  windows/aswsim.exe     - Windows executable
  macos/aswsim          - macOS executable
  linux/aswsim          - Linux executable

ARCHIVE FORMATS:
  Windows: .zip files
  macOS/Linux: .tar.gz files

USAGE:
  # Windows
  windows\aswsim.exe [options]
  
  # macOS
  ./macos/aswsim [options]
  
  # Linux
  ./linux/aswsim [options]

EXAMPLES:
  # Run with default settings
  ./aswsim

  # Run with custom parameters
  ./aswsim --n-targets 1000 --total-time 100 --velocity-type rayleigh

  # Run with custom position distribution
  ./aswsim --pos-mean-x 100 --pos-mean-y 200 --pos-std-x 50 --pos-std-y 30

  # Get help
  ./aswsim --help

EXTRACTION:
  # Windows (.zip)
  # Use built-in Windows Explorer or: tar -xf aswsim-windows.zip

  # macOS/Linux (.tar.gz)
  tar -xzf aswsim-macos.tar.gz
  tar -xzf aswsim-linux.tar.gz

VELOCITY TYPES:
  uniform: Uniform speed between min and max
  rayleigh: Rayleigh speed distribution
  beta: Beta distribution with min/max bounds
  bivariate-normal: Correlated velocity components
  independent-normal: Independent x/y velocities

PLATFORM SETUP:
  Windows: Double-click aswsim.exe or run from command prompt
  macOS: Make executable with 'chmod +x macos/aswsim' then run
  Linux: Make executable with 'chmod +x linux/aswsim' then run
